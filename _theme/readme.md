Use themes with both light and dark variants.

## Apps that support themes

Terminal apps:

- iterm2
- terminal.app

CLI apps:

- bat: a cat clone
- delta: diff viewer
- vim
- vivid: generates LS_COLORS

GUI apps:

- obsidian
- vscode: get using extensions
